# Pakistan Banking Dataset

This dataset contains 6000 customer records of Pakistani banks including HBL, UBL, and Meezan Bank.

## Features
- Customer demographics
- Account balance (PKR)
- Loan status
- Credit score
- Churn prediction

## Use Cases
- Machine Learning
- Data Analysis
- Banking Sector Research
